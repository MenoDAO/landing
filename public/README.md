# Logo Files

- `logo.svg` - Currently active SVG logo (temporary)
- `logo-reference.svg` - Reference SVG file

## To use your PNG logo:

1. Upload your PNG logo file and name it `logo.png`
2. Place it in this `/public/` directory
3. The code will automatically use it instead of the SVG

The code is configured to look for `/logo.png` or `/logo.svg` in the public directory.
